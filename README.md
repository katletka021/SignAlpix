# SignAlpix-beta

**Автор:** @geto320

Мощный менеджер подписи IPA с поддержкой всех версий iOS от 15.0 до 26.4

## 📱 Возможности

- Подпись IPA с менеджером сертификатов
- Выбор версии iOS 15.0 - 26.4 с таймером
- Файловый менеджер
- AppStore и Приложения
- Интеграция с Telegram @geto320
- Бело-красный интерфейс

## 📦 Установка

### Через GitHub Releases
1. Перейдите в [Releases](https://github.com/geto320/SignAlpix-beta/releases)
2. Скачайте последнюю версию `.ipa`
3. Установите через AltStore, ESign или SideStore

### Через сборку
```bash
git clone https://github.com/geto320/SignAlpix-beta.git
cd SignAlpix-beta
npm install
